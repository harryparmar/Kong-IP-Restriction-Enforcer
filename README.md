# Kong-IP-Restriction-Enforcer 
Python Program to enforce Private IPv4 address spaces to APIs on Kong API Gateway server and email If server is down. Open source community has given us some great tools like Kong API Gateway (https://github.com/Kong/kong)  and Konga (https://github.com/pantsel/konga), now is my turn to contribute. Email functionality and additional features will be added in later commits.
